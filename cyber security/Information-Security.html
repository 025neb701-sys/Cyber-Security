All about Information Security

Information Security (often called InfoSec) refers to the practice of protecting information — in both digital and physical forms — from unauthorised access, use, disclosure, disruption, modification, or destruction. It goes beyond just computers and networks; it encompasses printed documents, verbal communication, and any medium that carries valuable information. The key objectives of information security directly mirror the CIA Triad, but InfoSec as a discipline provides the policies, procedures, technologies, and human practices that actually implement and enforce those objectives across an organisation or individual's life. Information security operates across multiple layers. Physical security prevents unauthorised persons from physically accessing servers or documents. Network security uses firewalls, VPNs, and intrusion detection systems to guard digital pathways. Application security ensures that software is written and configured safely. Operational security (OPSEC) governs processes — like how employees handle data and who gets access to what. For individuals, information security means practices as simple as using strong passwords, locking your screen when away, not sharing sensitive information over unencrypted channels, and keeping software updated to patch vulnerabilities. ★Importance In Nepal and globally, data breaches cost organisations millions and destroy trust. For individuals, a single security lapse — like reusing a password — can lead to identity theft, financial loss, or reputational damage. InfoSec is everyone's responsibility, not just the IT department's. Physical Security Example: A company locks its server room with biometric access. Only authorised IT staff can enter, preventing physical theft of hard drives. Network Security Example: An employee working from home uses a VPN (Virtual Private Network) to connect to the office network. This encrypts data in transit, so even if someone intercepts the traffic, they cannot read it. Personal InfoSec Example: Riya uses a password manager to generate and store unique, complex passwords for every website. Even if one site is breached, her other accounts remain safe.
